# Pokedex

## Ejecutar en desarrollo
1. Clonar Repositorio

```
git clone ....
```

2. Instalar Dependencias

```
npm install
```

3. Tener Nest CLI

```
npm i -g @nestjs/cli
```

4. Levantar Base de Datos

```
docker-compose up -d
```

## Stack Utilizado
* MongoDB
* Nest
